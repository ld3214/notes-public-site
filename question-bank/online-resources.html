在线八股和刷题网站

这页集中放 ASIC/DV 复习时可以随手打开的在线资料：教程、八股、DDR 专项、题库和 OJ。使用时不要从上到下全刷，先按当前复习目标选入口。

快速使用

目标 先打开
补 Verilog / SV / UVM 基础 ChipVerify / VLSI Verify
刷 Verilog 手撕题 USTC Verilog OJ
准备 DDR4 project 追问 DDR 控制器设计完全指南 / DDR PHY 科普
找数字 IC 学习路线 Digital IC Learning Resources
看私有资料包 飞书资料夹

网站清单

网站 类型 适合用来做什么 复习建议
VLSI Verify Verilog / SV / UVM / Protocol 教程、quiz、interview questions 快速查 Verilog、SystemVerilog、UVM、coverage、assertion、RAL、AXI/APB 等八股 面试前按专题刷 interview questions；不熟的概念回到本地 SystemVerilog / UVM 和 Protocols
ChipVerify Digital design、Verilog、SV、UVM、verification 教程和在线 lab 查语法、UVM class、constraint、coverage、SVA、basic design examples 适合当速查手册；不要整站通读，遇到问题再查
USTC Verilog OJ Verilog 在线刷题 / OJ 练习 RTL 手撕题、组合逻辑、时序逻辑、FSM、计数器、握手类题目 每天挑 2-3 题，做完把容易忘的模式沉淀到 ASIC Frontend 手撕题
Digital IC Learning Resources 数字 IC 学习路线 / 课程 / 书籍 / 资源索引 找数字前端、Verilog、体系结构、课程和书籍路线 用来补学习地图，不要替代本地面试笔记；看到好资源再归档到对应知识库
DDR 控制器设计完全指南 DDR controller 系统教程 补 DDR controller 架构、PHY、timing、QoS、低功耗、验证策略 和 DDR4 Controller 复习地图 配合看，重点补 controller 后端行为
DDR PHY 科普 DDR PHY / PUB / training 科普 理解 DDR controller 和 PHY 的分工、DFI、training、DQ/DQS、delay line 等概念 面试只讲到自己项目边界：不要把 DDR PHY 说成自己实现的完整 RTL
飞书资料夹 私有资料夹 存放在线八股或课程资料，可能需要登录/权限 需要时打开查原始材料；重要内容再摘到本地笔记，避免复习时依赖网页权限

刷题节奏

周期 动作
每天 20 分钟 做 Verilog OJ 题，优先 FSM、counter、priority encoder、valid-ready、FIFO 类题
每天 20 分钟 刷 VLSI Verify / ChipVerify 的 SV-UVM 高频问答，闭卷口述 60 秒
每周 1 次 从 DDR controller / DDR PHY 资料里挑一个概念，补到 DDR4 项目复习地图或协议知识库

归档规则

  • 真正高频八股：整理到 question-bank.md。
  • 成体系知识点：整理到 Knowledge Base。
  • DDR4 项目相关：整理到 DDR4 Controller 复习地图。
  • 手撕题模式：整理到 ASIC Frontend 手撕题。
