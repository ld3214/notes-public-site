SystemVerilog / UVM

术语 全称 简短解释 相关笔记
UVM Universal Verification Methodology 基于 SystemVerilog 的验证方法学 UVM 总览
TLM Transaction-Level Modeling UVM 中组件之间传递 transaction object 的通信机制 UVM TLM 通信
Factory UVM Factory UVM 中用于对象创建和类型替换的机制 UVM Factory 机制
Objection UVM Objection 控制 run phase 是否可以结束的机制 UVM Phase 机制
RAL Register Abstraction Layer UVM 中用于抽象 DUT register map、统一寄存器访问和维护 mirror/check 的机制 Register Model / UVM RAL
Virtual Sequence UVM Virtual Sequence 协调多个 agent sequence 的系统级 sequence，常用于多接口场景 Virtual Sequence / Virtual Sequencer
Virtual Sequencer UVM Virtual Sequencer 保存 sub-sequencer、RAL model 和 config 等共享 handle 的 env 级协调器 Virtual Sequence / Virtual Sequencer
m_sequencer UVM sequence built-in sequencer handle sequence 被 start() 启动后由 UVM 设置的通用 sequencer handle m_sequencer vs p_sequencer
p_sequencer UVM typed sequencer handle 通过 `uvm_declare_p_sequencer 从 m_sequencer cast 出来的强类型 handle m_sequencer vs p_sequencer
Dynamic Array 动态数组 运行时分配长度的连续数组 动态数组、关联数组、队列
Associative Array 关联数组 以 key/value 方式存储的稀疏数组 动态数组、关联数组、队列
Queue 队列 支持 push/pop 的有序动态集合 动态数组、关联数组、队列
Implicit Conversion 隐式转换 编译器根据赋值、表达式或参数传递自动做的类型转换 Cast
Explicit Conversion 显式转换 工程师写出目标类型的转换，如 byte'(i)、$signed(x)、$cast() Cast
fork/join_none SV 并发语句 启动并发进程后立即继续执行，后续需要 wait fork 或其他同步方式等待结果 fork/join_none 行并行统计
valid-ready 握手协议 valid && ready 同周期为 1 时一次 transaction 完成 valid-ready task TB
Upcast 向上转换 派生类 handle 赋给基类 handle，通常隐式合法 Class Cast
Downcast 向下转换 基类 handle 转回派生类 handle，需要 $cast() 做运行时类型检查 Class Cast

Protocol / Interface

术语 全称 简短解释 相关笔记
APB Advanced Peripheral Bus AMBA 低功耗低速总线，无 outstanding，用于外设寄存器访问 APB 协议
UART Universal Asynchronous Receiver/Transmitter 异步串行通信接口，start bit + data + parity + stop bit 格式 APB-UART 验证环境
AXI VIP AXI Verification IP 可复用的 AXI 协议验证组件，提供 master/slave agent 和 protocol checker TVIP-AXI Crossbar 验证
Crossbar AXI Crossbar 多 master 到多 slave 的 AXI 互联结构，含地址解码和 arbitration TVIP-AXI Crossbar 验证
Gray Code 格雷码 相邻数值仅一位变化的编码，用于跨时钟指针同步 Async FIFO UVM 验证

Architecture

术语 全称 简短解释 相关笔记
OoO Out-of-Order Execution 指令可乱序执行、通常按序提交的微架构执行方式 OoO 基础
ROB Reorder Buffer OoO 中记录 in-flight 指令并按序提交的结构 OoO 基础
RS Reservation Station OoO 中等待操作数 ready 并选择指令发射到功能单元的结构 OoO RISC-V 处理器项目
PRF Physical Register File register renaming 后存放 physical register value 的寄存器堆 OoO RISC-V 处理器项目
LSQ Load-Store Queue OoO 中处理 load/store 顺序、forwarding 和 memory dependency 的结构 OoO 基础
MSHR Miss Status Handling Register nonblocking cache 中追踪未完成 cache miss 的结构 OoO RISC-V 处理器项目
BTB Branch Target Buffer branch predictor 中记录 branch PC 与 target PC 的表 OoO RISC-V 处理器项目
Cache Cache Memory CPU 和主存之间的小容量高速存储，用于降低平均访存延迟 Cache 基础
Cache Line Cache Block cache 和 memory 之间搬运的基本单位 Cache 基础
AMAT Average Memory Access Time 平均访存时间，常用 hit_time + miss_rate * miss_penalty 表示 Cache 基础
MESI Modified/Exclusive/Shared/Invalid 常见 cache coherence protocol 状态集合；E 状态支持 silent upgrade Cache 基础
MOESI Modified/Owned/Exclusive/Shared/Invalid MESI 扩展，O 状态允许 dirty line 直接分享而不写回 memory Cache 基础
TSO Total Store Order x86 的 memory consistency model，仅允许 store-load 重排 Cache 基础
Memory Barrier Fence 强制 CPU/编译器不跨越此点重排 memory operation Cache 基础
Snooping Snooping Protocol cache controller 监听共享总线上所有事务来维护 coherence Cache 基础
Directory Protocol Directory-based Coherence 用集中目录记录各 cache 副本状态，支持大核数扩展 Cache 基础
